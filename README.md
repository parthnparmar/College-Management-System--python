# College Management System-Python

A simple desktop application for college management built with Python and Tkinter.

## Features

- User authentication system
- Admin panel interface
- Database connectivity
- Clean GUI design

## Requirements

- Python 3.x
- tkinter (usually included with Python)
- Database connector (MySQL/PostgreSQL)

## Installation

1. Clone or download the project
2. Install required dependencies:
   ```bash
   pip install mysql-connector-python
   ```
3. Set up your database configuration in `db_config.py`
4. Run the application:
   ```bash
   python main.py
   ```

## Usage

1. Launch the application
2. Enter your username and password
3. Click "Login" to access the admin panel

## Project Structure

- `main.py` - Main login interface
- `admin_panel.py` - Admin panel functionality
- `db_config.py` - Database configuration
